| Problem                            | Solution                              |
| ---------------------------------- | ------------------------------------- |
| OLED is blank                      | Check SDA/SCL connections             |
| Motors don't move                  | Check DRV8833 power and motor wiring  |
| ESP32 resets                       | Check battery and power supply        |
| Wi-Fi network not visible          | Restart the ESP32                     |
| Robot moves in the wrong direction | Reverse motor polarity or update code |
